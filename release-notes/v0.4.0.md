# Путешествия на Kufar

## Коротко

Расширение теперь работает и с разделом путешествий на `travel.kufar.by`. Цены в этом разделе можно видеть в выбранной валюте так же, как на других поддерживаемых страницах Kufar. Также расширение аккуратнее сохраняет пометку «от» рядом с ценой, чтобы смысл предложения не терялся после конвертации.

---

## Что изменилось

### Для пользователей Kufar

* **Раздел путешествий:** добавлена поддержка `travel.kufar.by`. Теперь цены на страницах путешествий могут отображаться в выбранной валюте.
* **Цены «от»:** если цена была указана как «от 100 BYN», после конвертации пометка «от» остается на месте. Это помогает не потерять важный смысл цены.
* **Больше страниц раздела:** проверена работа на главной странице раздела, странице фильтра и страницах отдельных предложений.

### Работа в браузере

* **Версия 0.4.0:** обновлен номер версии расширения и подготовлены свежие ZIP-пакеты для Firefox и Chrome-based браузеров.
* **Свежие скриншоты:** добавлены скриншоты для версии 0.4.0, чтобы материалы релиза соответствовали текущему состоянию расширения.

---

## Исправили

* Исправили ситуацию, когда пометка «от» могла пропадать при замене цены — теперь она сохраняется рядом с конвертированной суммой.

---

## Технически

* Добавлены сохраненные примеры страниц `travel.kufar.by` и проверки конвертации для них. Это снижает риск случайно сломать поддержку раздела путешествий в следующих изменениях.
* Добавлена общая команда упаковки, которая собирает пакеты для Firefox и Chrome-based браузеров вместе.

---

## Что это значит для пользователя

Если вы смотрите предложения в разделе путешествий Kufar, расширение теперь должно показывать цены в выбранной валюте и сохранять важные подписи вроде «от». Обновление также подготовлено как отдельная версия с актуальными пакетами для установки.

---

## Полный список изменений

* Сохранена пометка «от» при конвертации цен.

# Kufar Travel Support

## Summary

The extension now works with the travel section at `travel.kufar.by`. Prices in this section can be shown in the selected currency, just like on other supported Kufar pages. The extension also preserves the "from" price label more carefully, so the meaning of the offer is not lost after conversion.

---

## What changed

### For Kufar users

* **Travel section:** added support for `travel.kufar.by`. Prices on travel pages can now be displayed in the selected currency.
* **From prices:** if a price was shown as "from 100 BYN", the "from" label remains after conversion. This helps preserve the important meaning of the price.
* **More section pages:** behavior was checked on the section home page, filter page, and individual offer pages.

### Browser experience

* **Version 0.4.0:** updated the extension version and prepared fresh ZIP packages for Firefox and Chromium-based browsers.
* **Fresh screenshots:** added screenshots for version 0.4.0 so release materials match the current extension state.

---

## Fixed

* Fixed a case where the "from" label could disappear when a price was replaced — now it stays next to the converted amount.

---

## Technical notes

* Added saved `travel.kufar.by` page examples and conversion checks for them. This reduces the risk of accidentally breaking travel section support in future changes.
* Added one package command that builds both Firefox and Chromium-based browser packages together.

---

## What this means for users

If you browse travel offers on Kufar, the extension should now show prices in your selected currency and keep important labels like "from". The update is also prepared as a separate version with current install packages.

---

## Full changelog

* Added support for `travel.kufar.by`.
* Preserved the "from" label when converting prices.
* Added checks using saved travel section pages.
* Updated the extension version to `0.4.0`.
* Prepared updated ZIP packages and screenshots for the release.
