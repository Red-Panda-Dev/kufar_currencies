# Легче и аккуратнее

## Коротко

Расширение стало значительно легче — пакет уменьшился примерно в 20 раз за счёт удаления лишних файлов из сборки. Курсы и конвертер работают как прежде, но устанавливать и обновлять расширение стало проще. Разрешения стали точнее — расширение запрашивает доступ только к тем адресам, которые действительно нужны.

---

## Что изменилось

### Для пользователей Kufar

* **Больше доменов:** недвижимость (re.kufar.by) и путешествия (travel.kufar.by) теперь поддерживаются наравне с авто. Цены конвертируются в выбрнную валюту на всех трёх доменах.
* **Разрешения стали точнее:** расширение обращается только к поддоменам Kufar и API курсов НБРБ — лишних разрешений больше нет.

### Работа в браузере

* **Пакет стал в разы легче:** ZIP-файл расширения уменьшился с ~1 МБ до ~45 КБ. Установка и обновление проходят быстрее, место в браузере экономится.
* **Сборка для Firefox и Chrome:** оба варианта собираются одинаково аккуратно, без лишних файлов внутри.

---

## Технически

* **Убраны лишние файлы из пакета:** raw-иконка (4 МБ) и пример ответа API больше не попадают в ZIP. Это никак не влияет на работу расширения — эти файлы нужны только разработчикам.
* **Разрешения в manifest.json:** убран дублирующийся паттерн `kufar.by/*`, так как `*.kufar.by/*` уже покрывает все поддомены.
* **Документация:** README и AGENTS.md приведены в соответствие с текущей структурой проекта и новыми командами сборки.

---

## Что это значит для пользователя

Расширение делает то же самое — показывает цены в удобной валюте на Kufar — но теперь его пакет весит гораздо меньше. Установка занимает меньше времени, а разрешений у расширения ровно столько, сколько нужно.

---

## Полный список изменений

* Пакет расширения уменьшен с ~1 МБ до ~45 КБ за счёт исключения raw-иконок и примеров API из сборки
* Убраны лишние host_permissions — остались только API НБРБ и поддомены Kufar
* Поддержка re.kufar.by и travel.kufar.by зафиксирована в документации как активная
* README обновлён: новые команды сборки, актуальная структура проекта, скриншоты
* AGENTS.md обновлён под структуру src/ и новые пути к файлам
* Версия в manifest.json обновлена до 0.5.0

# Lighter and Cleaner

## Summary

The extension package became roughly 20 times smaller by removing unnecessary files from the build. Exchange rates and the converter work exactly as before, but installing and updating the extension is now faster. Permissions are tighter — the extension only requests access to the addresses it actually needs.

---

## What changed

### For Kufar users

* **More domains:** real estate (re.kufar.by) and travel (travel.kufar.by) are now supported alongside auto. Prices convert to your chosen currency across all three domains.
* **Tighter permissions:** the extension only accesses Kufar subdomains and the NBRB exchange-rate API — no extra permissions.

### Browser experience

* **Much smaller package:** the extension ZIP went from ~1 MB to ~45 KB. Installation and updates are faster, and less browser storage is used.
* **Consistent Firefox and Chrome builds:** both variants are packaged cleanly without leftover files.

---

## Technical notes

* **Removed unnecessary files from package:** the raw icon (4 MB) and API response example no longer end up in the ZIP. This does not affect extension functionality — those files are only useful for developers.
* **Manifest permissions cleaned up:** the redundant `kufar.by/*` pattern was removed since `*.kufar.by/*` already covers all subdomains.
* **Documentation updated:** README and AGENTS.md brought in line with the current project structure and new build commands.

---

## What this means for users

The extension does the same thing — shows prices in your preferred currency on Kufar — but the package is now much smaller. Installation takes less time, and the extension has exactly the permissions it needs, nothing more.

---

## Full changelog

* Extension package reduced from ~1 MB to ~45 KB by excluding raw icons and API examples from the build
* Removed redundant host_permissions — only NBRB API and Kufar subdomains remain
* re.kufar.by and travel.kufar.by support confirmed as active in documentation
* README updated: new build commands, current project structure, screenshots
* AGENTS.md updated for src/ layout and new file paths
* Manifest version bumped to 0.5.0
